# selform-charter
The canonical English charter for Selform: a philosophical conversation about agency, truth, and self-formation that explains how technology may support rather than supplant human judgment.

## Positioning
- **Anti-product**: Selform is not a product brief—no MVP, feature list, or growth agenda is defined in this repository.
- **Anti-“user”**: Relationships here are described using terms such as *participant*, *subject*, and *practitioner*; avoid “user” unless discussing why that label misleads.
- **Anti-implementation**: This repository holds concept framing, principles, risks, and questions; it does not propose or describe an engineering implementation or roadmap.
