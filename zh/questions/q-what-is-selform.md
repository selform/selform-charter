# 问题：selform 究竟应被理解为何物？

- 已知立场：
  - 认知环境：承载问题与判断演化。
  - 伦理制度：宪法层 + 阈值层 + 审查层。
  - 关系性同伴：以语言共演化的第二人格。
- 进步信号：找到一个稳定术语，能明确治理义务，又不会把讨论拉回产品逻辑。
