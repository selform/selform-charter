# 概念：程序性摩擦
**定义**：在决策过程中有意引入的、具有意义的摩擦，用来保护判断形成条件（尤其在高情绪与高不可逆后果时）。  
**边界**：不是强制；不是锁死选择；不是伪装成关怀的 nudges；它保护程序完整性，而非追求特定结论。  
**重要性**：提供一种在避免不可逆伤害的同时，仍不让 agent 悄然接管主体意志的路径。  
**关联概念**：拒绝权；非支配；反操纵；预授权；审查层。  
**开放问题**：摩擦何时越界成操纵？哪些应进宪法层、哪些可下放阈值层？
