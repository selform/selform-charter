# 为什么是 selform
Selform 最初的种子来自一个务实的想法：如何把 AI agent 的能力深度融入笔记与学习管理的日常实践之中。但大多数“笔记 + AI”功能仍是旧范式上的连续性改良（总结、改写、问答），它们并未真正触及 agent 将成为第二大脑所带来的断裂。

Selform 用来命名这场断裂：当 agent 成为“第二大脑”时，它与第一大脑的关系不是清晰的分工，而是会通过语言发生深度耦合与交互——共同重塑记忆、判断，乃至我们对“人”的工作性定义。这种耦合需要一种新的技术伦理范式来理解与约束。

因此，这份宪章试图以 build in public 的方式推动该范式的生成：Selform 应被视为持续生成的伦理思想场域，而非应用；仓库记录概念演化与已形成的承诺，并在理想主张与可实践治理之间保持张力。
