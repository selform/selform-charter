# 决议：默认避免“用户”框架
- 决议：本仓库默认不把与 Selform 发生关系的人称为“用户”；优先用“参与者 / 实践者 / 主体”。
- 背景：“用户”隐含消费式主客关系，容易把讨论拉回传统产品逻辑。
- 理由：Selform 的讨论以主体性与自我形成为核心；词汇选择应守护这一框架。
- 影响：模板与文档应默认采用非“用户”表述，除非明确在讨论“为何用户框架会误导”。
