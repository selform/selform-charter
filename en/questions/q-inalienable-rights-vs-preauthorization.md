# Question: Which rights are inalienable vs pre-authorizable?
- Why open: Pre-authorization can make friction “delayed self-governance,” but it does not erase manipulation; not all rights should be delegable.
- Known positions:
  - Constitution layer freezes a thin set of non-negotiables (no lying, no hidden manipulation, refusal/exit rights, no taking over irreversible decisions).
  - Threshold layer supports revocable, individualized permissions.
- Progress signals: A clear boundary list + an interface that prevents pre-authorization flows from becoming covert manipulation.
